# Urbanity – Figma Designs

This folder contains the UI/UX design files for the **Urbanity Project**.

The designs were created using **Figma** and include the complete interface layouts for the different user roles in the system.

## Included Design Screens
- Authentication Pages (Login / Signup)
- Citizen Dashboard
- Department Head Dashboard
- Department Officer Dashboard
- Field Worker Interface
- Admin Interface
